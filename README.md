O **LifeBoard Frontend** é a interface web do LifeBoard, desenvolvida com **React + Vite + TypeScript**, oferecendo uma experiência moderna, intuitiva e responsiva. Ele integra diretamente com a API do LifeBoard para fornecer controle financeiro, gestão de metas, organização de tarefas e ferramenta Pomodoro.

---

## 🚀 Tecnologias Utilizadas

- **React.js**
- **Vite**
- **TypeScript**
- **React Router DOM**
- **Styled-Components**
- **Recharts**
- **Swiper**
- **jwt-decode**
- **ExcelJS + FileSaver**
- **Axios**

---

## 🔐 Principais Funcionalidades

- ✔️ Autenticação com JWT (login e cadastro)
- ✔️ Dashboard financeiro com saldo, salário e gastos do mês
- ✔️ Histórico de gastos em gráfico
- ✔️ Listagem completa de transações com filtros e exportação para XLSX
- ✔️ Sistema de metas financeiras com acompanhamento visual
- ✔️ Gestão de tarefas no formato Kanban e tabela
- ✔️ Exportação de tarefas para XLSX
- ✔️ Pomodoro integrado (5, 15 e 20 minutos)
- ✔️ Layout responsivo e pronto para produção

---

## ⚙️ Configuração do Ambiente

**🔧 Pré-requisitos**

Certifique-se de ter instalado:

- Node.js 18+
- NPM ou Yarn

---

## 📁 Configuração do `.env`

Na raiz do projeto, crie ou configure o arquivo:

```bash
VITE_API_URL=http://localhost:8080
```

A variável acima deve apontar para a API do backend do LifeBoard.

---

## ▶️ Como Rodar o Projeto

**Instalando dependências**

```bash
npm install
```

Ou:

```bash
yarn
```

**Executando em modo desenvolvimento**

```bash
npm run dev
```

Aplicação disponível em:

[http://localhost:5173](http://localhost:5173)
